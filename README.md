### Local RAG News Intelligence System
Автономный AI-конвейер для сбора, индексации и аналитической обработки новостных потоков. Система работает полностью локально: парсит RSS-ленты в реальном времени, извлекает полный текст статей, векторизует контент и отвечает на сложные аналитические вопросы пользователя через RAG-архитектуру с Chain-of-Thought промптингом.

### Архитектура и ключевые решения
Параллельный пайплайн сбора данных. Фоновый поток непрерывно парсит RSS, дедуплицирует статьи и накапливает векторное хранилище. Основной поток обрабатывает пользовательские запросы без блокировок.
Продвинутая RAG-оркестрация. Реализован кастомный Few-Shot + CoT промпт с эталонными примерами. LLM строго ограничена ретривленным контекстом — нет галлюцинаций, нет внешних API.
Оптимизация под локальное железо. Работа с моделями Qwen 2.5 7B и BGE-M3 через Ollama на CPU без облачных зависимостей. Полная приватность данных.
#### Результат
Система превращает неструктурированные новостные потоки в структурированную базу знаний с возможностью сложного семантического поиска и аналитических рассуждений — end-to-end, offline, zero cloud costs.


### Стек

- **Python 3.14** — основная среда разработки
- **LangChain** — фреймворк для работы с текстовыми сплиттерами и построения RAG
- **Ollama** — локальный сервер для запуска языковых моделей
- **BeautifulSoup4** — парсинг HTML-страниц и извлечение текста новостей
- **ChromaDB** — векторная база данных для хранения и поиска семантических соответствий
- **Requests** — загрузка данных с веб-сайтов и RSS-лент
- **Few-Shot CoT Prompting**
- **Multithreading**

### Возможности

- Автоматический парсинг в отдельном потоке 
- Дедупликация статей
- Разбиение на чанки с перекрытием
- Локальное векторное хранилище
- Консольный интерфейс для вопросов и ответов
- Полная приватность — все данные и модель работают локально
